### 常见错误及解决方案

#### 错误 1: ECONNREFUSED
```
[MQTT] ❌ 错误: connect ECONNREFUSED 127.0.0.1:3333
```

**原因**：SharkDataSever 未启动或端口错误

**解决**：
1. 确认 SharkDataSever 已启动
2. 确认端口是 3333
3. 检查防火墙设置

#### 错误 2: ETIMEDOUT
```
[MQTT] ❌ 错误: connect ETIMEDOUT
```

**原因**：连接超时

**解决**：
1. 检查网络连接
2. 检查防火墙是否阻止
3. 尝试增加超时时间

#### 错误 3: ENOTFOUND 或 getaddrinfo
```
[MQTT] ❌ 错误: getaddrinfo ENOTFOUND localhost
```

**原因**：DNS 解析失败（这就是为什么改用 127.0.0.1）

**解决**：
1. ✅ 使用 IP 地址 `127.0.0.1` 而不是 `localhost`
2. 或修复系统的 hosts 文件
